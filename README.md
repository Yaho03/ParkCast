# ParkCast

**AI 주차 수요 예측 기반 실시간 분산 추천 시스템**

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

http://localhost:5173/parkcast-demo/ 접속

## GitHub Pages 배포

```bash
npm run deploy
```

`https://<username>.github.io/parkcast-demo/` 에서 확인

## 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | React 19, Vite 6 |
| 배포 | GitHub Pages |
